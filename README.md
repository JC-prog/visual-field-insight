# VisualFieldInsight

VisualFieldInsight is an AI-powered OCR and data extraction tool for **visual field test reports**.

The goal of this project is to automatically extract structured data from visual field documents such as:

- Humphrey Visual Field (HVF) reports from the Zeiss Humphrey Field Analyzer
- Virtual Reality visual field test reports
- Other perimetry systems

This enables automated analysis, research pipelines, and integration with clinical data systems.

---

## Features

- OCR for visual field reports
- Structured extraction of key clinical metrics
- Support for multiple report formats
- Designed for Humphrey and VR-based perimetry systems
- Extensible pipeline for new document types

Future capabilities:

- Machine learning assisted parsing
- Longitudinal visual field progression analysis
- Structured dataset export
- Integration with research workflows

---

## Example Use Cases

- Automating extraction of visual field metrics
- Creating datasets for ophthalmology research
- Converting PDF reports into structured JSON
- Supporting AI models for glaucoma progression analysis
